🧠 What AI does per transaction

✓Classifies type — expense, income, investment, or transfer
✓Assigns category — matched to your existing category list
✓Cleans the description — "UPI/DR/771234/PETROLBUNK/OKAXIS" → "Petrol Bunk"
✓Gives a confidence score — shown as a colour-coded bar
✓Explains its reasoning — expandable "Why?" tooltip per row
✓Suggests a rule — "Create rule for this pattern?"

⚡ Performance & Cost

📦Batch processing — all unknowns sent in one API call, not one per row
🔢Typical batch size — 20–60 rows (rest handled by rules)
⏱️Processing time — 3–8 seconds for a full statement
💰Cost — ~₹0.10–0.30 per statement import (Haiku model)
🔄Cached — same merchant never sent twice; rule created automatically after first AI classification
🔒Privacy — only description text sent, never amounts or account numbers
